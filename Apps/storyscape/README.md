# Alex Chen - Professional Portfolio

A modern, Apple-inspired portfolio website showcasing the work and expertise of a senior frontend developer. Built with Next.js, TypeScript, and Framer Motion for smooth, professional animations.

## ✨ Features

- **Professional Design**: Apple-level design quality with clean, modern aesthetics
- **Scroll-based Animations**: Smooth parallax effects and section reveals using Framer Motion
- **Interactive Elements**: Hover effects, micro-interactions, and smooth transitions
- **Responsive Design**: Optimized for all devices and screen sizes
- **Performance Optimized**: React Suspense, lazy loading, and efficient animations
- **TypeScript**: Full type safety throughout the application

## 🚀 Technologies Used

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Advanced animations and scroll effects
- **React Intersection Observer** - Scroll-triggered animations

## 🛠️ Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open your browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)
