## Demo app (try it first)

To see if React Native feels smooth and native enough:

```bash
cd mobile
npm install
npm start
```

Then press **`i`** for iOS Simulator, or scan the QR code with **Expo Go** on your iPhone. Full steps: **[mobile/README.md](mobile/README.md)**.
