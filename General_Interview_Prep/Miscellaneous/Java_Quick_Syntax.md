# Quick Java Syntax

Use when a basic Java snippet is unexpectedly required.

```java
// Main Class
public class Main {
    public static void main(String[] args) {
        // Variables
        int x = 10;
        String name = "Example";

        // Loop
        for (int i = 0; i < 5; i++) {
            System.out.println("Count: " + i);
        }

        // Condition
        if (x > 5) {
            System.out.println("High");
        }
    }
}
```
