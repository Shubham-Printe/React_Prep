# DXC Technology — GCP Cloud Engineer / React Developer

Prep plan for the **DXC Technology** role (Project Cirrus) at a large global investment bank. The role focuses on FIC (Fixed Income and Currencies) cloud migrations, GCP infrastructure, and React-based UI development for compliance archive products.

## Company

DXC Technology is a unique blend of engineering excellence and deep industry expertise to serve clients globally, specializing in automotive, financial services, travel and hospitality, healthcare, life sciences, media and telecommunications.

## Role Summary

- **Title:** GCP Cloud Engineer / React Developer
- **Employer:** DXC Technology
- **Client:** Large global investment bank
- **Project:** Project Cirrus (FIC cloud migrations)
- **Stack (JD):** React JS, Redux, GCP (GCE, GKE, CloudSQL/Postgres), Terraform, REST APIs
- **Domain:** Compliance archiving, real-time communications data, regulatory requirements
- **Experience required:** 7–10 years (React and JavaScript)
