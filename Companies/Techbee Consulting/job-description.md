# TechBee Consulting – React JS Developer

**Role:** Front End Developer  
**Industry:** IT Services & Consulting  
**Department:** Engineering - Software & QA  
**Employment Type:** Full Time, Permanent

## Job Description

We are looking for a skilled React JS Developer to join our team at TechBee Consulting. The ideal candidate will have a strong background in React JS and be able to work on complex projects.

## Roles and Responsibilities

- Design, develop, and test React JS applications with high-quality code.
- Collaborate with cross-functional teams to identify and prioritize project requirements.
- Develop and maintain reusable components and libraries for future use.
- Troubleshoot and resolve technical issues efficiently.
- Participate in code reviews and contribute to improving overall code quality.
- Stay updated with the latest trends and technologies in React JS development.

## Job Requirements

- Strong proficiency in React JS, including Hooks, Context API, and Redux.
- Experience with modern frontend build tools such as Webpack, Babel, and ESLint.
- Knowledge of CSS preprocessors like Sass or Less.
- Familiarity with state management libraries like MobX or Recoil.
- Excellent problem-solving skills and attention to detail.
- Ability to work collaboratively in a team environment and communicate effectively.
