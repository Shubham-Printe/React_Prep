# Document submission — Zip structure + email + cover letter

Use this when sending your documents to Atlas Copco HR as a single zip file, with an explanatory email and a cover letter inside the zip.

---

## 1. Zip folder structure (create these folders, then zip)

Organize files **before** zipping so HR can open one folder and find everything in order.

```
Shubham_Printe_AtlasCopco_Documents.zip
│
├── 00_Cover_Letter.pdf                    ← Explains employer name changes & what each section contains
│
├── 01_Identity/
│   ├── Aadhaar.pdf
│   └── PAN.pdf
│
├── 02_Current_Employer_Digialpha/         ← One company; name changed twice (Hartley Lab → Betacraft → Digialpha)
│   ├── Company_Name_Change_Support.pdf    ← Official docs from company supporting the name changes
│   ├── Appointment_Letters/
│   │   ├── 01_Hartley_Lab_Intern_26Oct2021.pdf
│   │   ├── 02_Betacraft_Intern_27Oct2021.pdf
│   │   └── 03_Digialpha_Sr_Software_Engineer_01Jan2024.pdf
│   ├── Increment_Letters/                 ← All increment letters (different company names, same employer)
│   │   ├── [chronological, e.g. 01_YYYY_MM.pdf, 02_YYYY_MM.pdf ...]
│   ├── Salary_Slips_Last_3_Months/
│   │   ├── [e.g. Nov2024.pdf, Dec2024.pdf, Jan2025.pdf]
│   └── Salary_Compensation_Breakup.pdf    ← Or note: "As per salary slip" if no separate annexure
│
└── 03_Previous_Employer_Dogra_Technology/
    ├── Appointment_Letter.pdf
    ├── Salary_Slip(s).pdf                 ← One file or multiple, as you have
    └── Experience_Letter.pdf
```

**Naming tip:** Use clear, short names (e.g. `01_Hartley_Lab_Intern_26Oct2021.pdf`). Avoid spaces in filenames if possible; use underscores.

**Proof of bonus/performance:** If you have no separate document, add a one-page PDF in `02_Current_Employer_Digialpha/` titled e.g. `Proof_of_Bonus_Performance_NA.pdf` with a single line: “Not applicable / No separate bonus or performance incentive documentation for the period.” Or mention it only in the cover letter and skip the file.
