# Practice Checklist
- Solve 2 coding problems per week (arrays, objects, async)
- Build 1 mini feature per week and demo it (reusable component or small flow)
- Write or improve tests weekly
- Do 1 mock interview per week (technical + behavioral)
- Do 1 code review exercise per week (align with JD code quality focus)

---

## DSA sharpener (for odd array/object/async questions)
- **Arrays**: Two sum; flatten nested array; group by key; merge sorted arrays; find duplicates / unique values; chunk array into size N.
- **Objects / Maps**: Deep clone object; deep merge; count frequency (array → object); LRU-style “recent N items” (Map + order).
- **Strings**: Valid palindrome; first non-repeating character; simple anagram check (sort or count).
- **Async / Promises**: Sequential vs parallel execution; retry N times; debounce/throttle (timers + closure); Promise.all vs allSettled when one fails.
- **Linked list / tree (if they go there)**: Reverse a list; max depth of tree; level-order (BFS) traverse.
