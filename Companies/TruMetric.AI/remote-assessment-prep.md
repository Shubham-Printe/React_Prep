# Remote Proctored Technical Assessment – Preparation & Checklist

**Assessment type:** Online, fully proctored (camera + rules enforced)  
**Use this doc:** To prepare before the day and to follow on the day of the assessment.

---

## 1. Devices & setup (do this before assessment day)

### Laptop/Desktop
- [ ] Fully charged (or plugged in with cable visible/acceptable per instructions)
- [ ] Camera working and tested
- [ ] Microphone working and tested
- [ ] Stable desk/table; you will sit **facing the screen**
- [ ] Good internet; close heavy apps/streaming; prefer wired connection if possible

### Mobile phone
- [ ] Fully charged
- [ ] Camera and microphone working
- [ ] Plan where it will be placed: **stable surface** so it shows:
  - You
  - Your laptop screen
  - Desk area  
  (Tripod, stack of books, or phone stand can help.)

### Meeting
- [ ] Meet link saved (calendar invite / email)
- [ ] Same link will be used on **both** laptop and phone
- [ ] Join **at least 5 minutes early** for setup

---

## 2. Room & desk (day of)

- [ ] Desk **clear**: no books, notes, extra phones, tablets, or second monitors (unless allowed)
- [ ] No **headphones/earphones**
- [ ] Room quiet; no one else in frame or walking behind you
- [ ] Lighting good enough for both cameras (face visible, no strong backlight)

---

## 3. Rules to avoid disqualification

| Rule | Action |
|------|--------|
| Both cameras ON | Laptop + phone cameras ON from join until you’re told to leave |
| Do not change camera position | Set phone once and don’t move it |
| Start only after proctor approval | Wait for explicit “you can start” (or equivalent) |
| Desk clear | Only laptop, one phone, keyboard, mouse, water if allowed |
| No headphones | Use laptop/device speakers only |
| Do not leave camera view | No bathroom/breaks unless proctor allows |
| Laptop + phone on same Meet | Join same link on both devices as instructed |

---

## 4. Day-of timeline (example)

| Time | Action |
|------|--------|
| **T–15 min** | Desk cleared, devices charged, phone position set (you + laptop + desk in view) |
| **T–5 min** | Join the Meet link on **laptop** (camera + mic ON), then join **same link on phone** and position it |
| **T–0** | Wait for proctor; do not start assessment until they approve |
| **During** | Follow proctor instructions; if in doubt, ask in chat/camera before acting |

---

## 5. Quick technical prep (if assessment is coding/technical)

- [ ] Restart laptop before joining (fewer background apps)
- [ ] Close Slack, email, extra browsers, IDE unless you’re told to use a specific tool
- [ ] Know whether you’ll use: browser-only coding, shared IDE, or local IDE (and test it beforehand if allowed)
- [ ] Have a backup: mobile hotspot or alternate network in case Wi‑Fi drops

---

## 6. One-line reminder

**Join 5 min early on both devices → same Meet link → wait for proctor approval → cameras ON, desk clear, no headphones, don’t leave view.**

Good luck for Monday.
