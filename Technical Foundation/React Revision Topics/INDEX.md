# Senior React Topics: Index

This index organizes revision topics for senior-level React interviews and architectural discussions.

## Document Format Guide
To ensure consistency and clarity, all conceptual sections (where applicable) should follow this structure:
1. **Why is this needed?** (The problem statement or motivation)
2. **How it works** (Brief technical explanation or mental model)
3. **How it solves the problem** (The benefit/outcome)
4. **Exceptions & miscellaneous** (Edge cases, limitations, or alternatives)
